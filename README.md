# AudioButcher
AudioButcher is a program that randomizes segments of audio for a defined amount of time.

Features: 

= 3 scrambling methods,
= Backmasking (symmetrical and/or asymmetrical),
= Stumbling,
= Speed variations,
= Average start times,

...and total control over each parameter!
